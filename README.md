# 🎓 Udemy Clone Website

Welcome to **Udemy Clone** — a responsive front-end project built using **HTML5** and **CSS3** with **Font Awesome** icons.

This project replicates Udemy’s layout including a header, categories, banner, recommended courses, topics section, most popular courses, and a clean footer.

---

## 📄 Pages / Sections

* **Header** — Logo, search bar, navigation links (Courses, My Learning, Cart, Notifications, Profile)
* **Categories** — Quick buttons for Business, Development, Design, IT, Marketing, etc.
* **Banner** — Promotional section with sale details
* **Recommended** — Suggested course cards
* **Topics** — Popular tech topics (JS, MongoDB, AWS, GitHub, etc.)
* **Most Popular** — Grid of trending courses with ratings & prices
* **Footer** — Links (Business, Careers, Help, etc.) and copyright

---

## 🔑 Features

* Responsive layout with flexbox & grid
* Search bar with rounded styling
* Hover effects on buttons & navigation links
* Course cards with images, titles, ratings, and prices
* Modern UI with clean colors and shadows

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Font Awesome (icons)
